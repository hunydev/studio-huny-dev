작고 재밌는 웹앱들을 한 곳에

썸네일 갤러리에서 바로 열고, 태그로 필터링하고, 검색으로 찾아보세요. 가벼운 앱은 /app 아래에서 동작합니다.

데이터 관리 (GitHub → Netlify)

  1. apps.json 파일을 저장소 루트에 두면 이 페이지가 자동으로 불러옵니다. (예: /apps.json)
  2. 스키마: 
    {
  "links": [ { "title": "huny.dev", "url": "https://huny.dev" }, { "title": "blog.huny.dev", "url": "https://blog.huny.dev" } ],
  "apps": [
    {
      "id": "prompt-keeper",
      "title": "Prompt Keeper",
      "description": "AI 프롬프트 저장/검색",
      "url": "/app/prompt-keeper",          // 내부 경로면 studio.huny.dev/app/... 으로 열림
      "thumbnail": "/thumbs/prompt-keeper.webp", // 선택. 없으면 자동 생성 썸네일 UI 사용
      "tags": ["ai", "productivity"],
      "featured": true,                         // 선택. 추천 표시/정렬 가중치
      "updatedAt": "2025-08-10"                // 선택. 정렬에 사용
    }
  ]
}
  3. Netlify에 배포 시 apps.json만 수정해도 갤러리가 갱신됩니다.
